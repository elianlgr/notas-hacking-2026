## Descripción

- This tar file got tarred a lot.
## Solución

- descargamos el archivo con wget
- con el comando tar -tf vemos su contenido
- hacemos tar -xvf para desempaquetar un archivo dentro del primero 
- al arecer tenemos que hacer tar, desde el archivo 1000 hasta el archivo 1
- hacemos un ciclo for que vaya desde el archivo 1000 hasta el archivo 1 de esta manera automatizamos el proceso y encontramos la bandera
