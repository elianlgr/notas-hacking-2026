## Descripción

- Can you break into this super secure portal?
- nos entrega una liga para resolver el reto 
## Solución

- usamos la herramienta inspector para ver el codigo
- al haber código ofuscado usamos la herramienta js nice para hacerlos mas legible 
- nos centramos en un arreglo encontrado en el código
- lo ordenamos y así obtenemos la bandera
