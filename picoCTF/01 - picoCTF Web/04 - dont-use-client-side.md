## Descripción

- Can you break into this super secure portal?
- nos da una ligar para resolver el reto 
## Solución

- con la herramienta inspector entramos a analizar el código fuente
- analizándolo vamos que la bandera esta escondida dentro de texto plano, simplemente tenemos que ordenarla como si fueran las posiciones de un arreglo y tendremos la bandera  
## Notas adicionales 

- Client-side
	- el procesamiento toma lugar en la computadora del usuario y requiere que el navegador corra script sin necesidad de involucrar ningún procesamiento del lado del servidor 
- Server-side
	- el procesamiento toma lugar en el servidor
