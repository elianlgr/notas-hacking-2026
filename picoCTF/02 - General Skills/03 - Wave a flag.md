## Descripción

se solicita invocar solicitadores de ayuda para buscar la bandera, estos contienen informacion muy útil 
## Solución

- primero descargamos el archivo con el comando wget y el enlace del archivo 
- otorgamos los permisos necesarios para poder tener la información necesaria con el comando chmod
- con el comando file vemos el tipo de dato de warm 
- al ver que es un archivo ejecutable usamos ./ y nos aparece un mensaje indicándonos que usemos -h para mayor información o ayuda, al usarlo nos aparece la bandera
