## Descripción

- Unzip this archive and find the file named 'uber-secret.txt'
## Solución

- usamos el comando wget para descargar el archivo .zip 
- usamos el comando unzip para ver el contenido
- usamos el comando find . -name 
- el comando nos regresa una ruta para hacer la busqueda
- con el comando cat abrimos los archivos y encontramos la bandera
