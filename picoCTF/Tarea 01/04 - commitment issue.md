## Descripción

- I accidentally wrote the flag down. Good thing I deleted it!
## Solución

- Descargar el archivo challenge.zip que contiene el repositorio del reto.
    
- Descomprimir el archivo descargado utilizando el comando: unzip challenge.zip
    
- Navegar hacia el interior de la carpeta extraida utilizando el comando: cd drop-in
    
- Añadir una excepcion de seguridad en Git para el directorio actual utilizando el comando: git config --global --add safe.directory /home/kali/Hacking/Tarea1/drop-in
    
- Consultar el historial detallado de cambios del repositorio utilizando el comando: git log -p
    
- Localizar y copiar la bandera del reto dentro de las lineas de texto del historial que muestran los datos eliminados.
