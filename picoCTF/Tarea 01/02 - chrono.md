## Descripción

- How to automate tasks to run at intervals on linux servers?
## Solución

- Iniciar sesión en el servidor remoto por medio de SSH utilizando el comando: ssh -p 56424 picoplayer@saturn.picoctf.net
    
- Mostrar el contenido del archivo de configuración de tareas automáticas del sistema usando el comando: cat /etc/crontab
