## Descripción

- What was I last working on? I remember writing a note to help me remember...
## Solución

- Descargué el archivo challenge.zip que contiene los archivos del reto.
    
- Descomprimí el archivo descargado en la terminal utilizando el comando: unzip challenge.zip
    
- Ingresé a la nueva carpeta generada utilizando el comando: cd drop-in
    
- Añadí la excepción de seguridad en Git para el directorio actual utilizando el comando: git config --global --add safe.directory /home/kali/Hacking/Tarea1/drop-in
    
- Revisé el historial general de cambios para leer las notas guardadas utilizando el comando: git log
    
- Identifiqué y copié la bandera del reto que el creador escribió directamente dentro del mensaje de confirmación.
    
- Presioné la tecla q para salir de la vista del historial.
