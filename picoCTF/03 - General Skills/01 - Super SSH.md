## Descripción

- Using a Secure Shell (SSH) is going to be pretty important.
- al iniciar la instancia nos entrega los datos para conectaros al servidor 
## Solución

- usamos el comando ssh -p para conectarnos al servidor con los datos que nos entrega al indicar la instancia 
- al conectarnos nos da directamente la bandera
