## Descripción

- Download the disk image and use `mmls` on it to find the size of the Linux partition. Connect to the remote checker service to check your answer and get the flag.
- Note: if you are using the webshell, download and extract the disk image into `/tmp` not your home directory.
- nos da un archivo para descargar y poder resolver el reto 
## Solución

- primero descargamos la imagen de disco usando el comando wget y descomprimimos el archivo con el comando gzip
- usamos le comando mmls para conseguir mas informacion y resolver el reto 
- nos conectamos el servidor y nos pregunta cual es el tamaño de partición de linux en la imagen, copiamos y pegamos el resultado y terminamos
