## Descripción

- This file contains more than it seems.
- nos da un archivo para descargar en la terminal, es un archivo jpg y se tiene que hacer la búsqueda dentro de la imagen 
## Solución

- primero descargamos el archivo con el comando wget 
- le aplicamos un file para ver las propiedades y el tipo de archivo
- hacemos uso de una herramienta integrada de kali llamada " Hexeditor " podemos ver el archivo sin procesar
- usamos el comando Cntrl+W para hacer una búsqueda, usamos la opción de búsqueda por formato de texto y asi encontramos la bandera 
## Notas adicionales 

- otra forma de resolverlo es usar el comando Strings y el nombre del archivo y te dará todas las cadenas de texto
