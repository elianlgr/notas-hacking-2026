## Descripción

- This vault uses some complicated arrays! I hope you can make sense of it, special agent. The source code for this vault is here
## Solución
- descargué el archivo vaultdoor1.java desde el enlace proporcionado en el reto
- abrí el archivo de código fuente usando mi terminal para examinar su contenido
- analicé la función checkpassword y noté que valida la contraseña carácter por carácter usando el método charat()
- me di cuenta de que las posiciones de las letras estaban completamente desordenadas en el código
- ordené manualmente cada carácter guiándome por el número de índice que indicaba la función
