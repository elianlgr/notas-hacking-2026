## Descripción

- I have these 2 images, can you make a flag out of them?
## Solución

- instalar la librería de Python necesaria para manipular imágenes, que en este caso es Pillow.
- crear un script de Python en la misma carpeta donde tienes descargadas las dos imágenes del reto.
- programar el script para que tome ambas imágenes y sume sus valores de color píxel por píxel usando aritmética modular.
- correr el código para generar y guardar una nueva imagen con el resultado de esa suma.
- abrir la imagen generada para leer el texto de la bandera.
