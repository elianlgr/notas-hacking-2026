## Descripción

- nos dice que hacer una búsqueda en un archivo de texto demasiado grande es muy tedioso, pero hay una forma de hacerlo más eficiente 
## Solución

- primero descargamos el archivo de texto usando el comando wget y el enlace del archivo 
- al tenerlo descargado usamos el comando file para obtener información de los tipo de datos que contiene el archivo 
- al ser un archivo de texto demasiado extenso usamos el comando grep para hacer una búsqueda con una palabra clave, en este caso fue "pico"
