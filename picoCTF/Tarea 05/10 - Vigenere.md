## Descripción

- Can you decrypt this message?
## Solución

- Leí el contenido del archivo cipher.txt usando el comando cat cipher.txt en la terminal.
    
- Extraje la cadena cifrada rgnoDVD{O0NU_WQ3_G1G3O3T3_A1AH3S_2951c89f}.
    
- Identifiqué mediante la descripción del reto que el texto estaba protegido con un cifrado Vigenere utilizando la llave CYLAB.
    
- Utilicé la herramienta web CyberChef con la operación Vigenere Decode, introduciendo la llave CYLAB y el texto cifrado.
