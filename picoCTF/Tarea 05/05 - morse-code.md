## Descripción

- Morse code is well known. Can you decrypt this?
- Wrap your answer with picoCTF{}, put underscores in place of pauses, and use all lowercase.
## Solución

- Utilicé una herramienta web de decodificación para automatizar la extracción del código Morse desde el archivo de audio.
    
- La herramienta arrojó una transcripción aproximada (`AEH47 H47H 90D W20U9H7`) debido a las variaciones e imperfecciones en los espacios del sonido.
    
- Analicé el resultado parcial y, utilizando análisis de contexto histórico e iteraciones de caracteres, deduje que el mensaje original codificado en Leetspeak era `WH47 H47H 90D WR0U9H7` ("What hath God wrought").
