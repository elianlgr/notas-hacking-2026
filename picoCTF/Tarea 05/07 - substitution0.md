## Descripción

- A message has come in but it seems to be all scrambled. Luckily it seems to have the key at the beginning. Can you crack this substitution cipher?
## Solución

- Ejecuté el comando cat message.txt para visualizar el contenido completo del archivo desde la terminal.
    
- Identifiqué que el documento contenía un texto ofuscado mediante un cifrado de sustitución simple.
    
- Noté que la primera línea del archivo funcionaba como la llave del cifrado, revelando el alfabeto modificado.
    
- Mapeé los caracteres de la llave contra el alfabeto estándar para conocer el valor original de cada letra.
    
- Apliqué la sustitución inversa a la última línea del texto para descifrar el mensaje oculto respetando los caracteres numéricos.
