## Descripción

- Run the Python script and convert the given number from decimal to binary to get the flag
## Solución

- ejecute el comando wget seguido del enlace para descargar el archivo convertme.py

- ejecute el comando python3 convertme.py para iniciar el script y ver el numero decimal asignado

- ejecute el comando python3 -c "print(bin(43))" en otra terminal para realizar la conversion del numero a binario

- ingrese el resultado 101011 en la terminal original que estaba en espera para resolver el reto y obtener la bandera
