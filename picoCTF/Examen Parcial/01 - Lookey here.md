## Descripción

- Attackers have hidden information in a very large mass of data in the past, maybe they are still doing it.
## Solución

- ejecute el comando wget [https://artifacts.picoctf.net/c/126/anthem.flag.txt](https://www.google.com/search?q=https://artifacts.picoctf.net/c/126/anthem.flag.txt) para descargar el archivo de texto
- liste los archivos del directorio con el comando ls para confirmar que se descargo correctamente
- ejecute el comando grep picoctf anthem.flag.txt para filtrar y encontrar la cadena de texto de la bandera
