## Descripción

- I've hidden a flag in this file. Can you find it?
- nos da un archivo para descargar y resolver el reto 
## Solución

- descargamos el archivo con el comando wget y apóstrofos porque tiene algunos espacios 
- le aplicamos un file para ver que tipo de archivo es (pp 2007)
- desempaquetamos con un unzip y vamos al explorador de archivos para ver el contenido 
- usamos visual studio code para usarlo como explorador de archivos y ahí encontramos la bandera pero en código 64
- hacernos la conversión y terminamos
